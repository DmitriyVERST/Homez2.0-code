# Homez — Каталог аренды жилья

# Live: https://dmitriyverst.github.io/Homez2.0/

---

## Описание проекта

Проект **Homez** — это современный и удобный каталог аренды жилья, созданный для быстрого поиска и бронирования недвижимости. Он предназначен для облегчения процесса поиска жилья для арендаторов и повышения доверия клиентов.

На сайте реализованы привлекательные секции с изображениями, отзывами и фильтрами поиска. Использование библиотек **Swiper** для слайдеров и **AOS** для анимаций делает интерфейс динамичным и интерактивным. Чистый и современный дизайн обеспечивает комфортное взаимодействие на всех устройствах.

---

## Используемые технологии

- **HTML**, **CSS**, **JavaScript** — основы веб-разработки  
- **Swiper** — для создания отзывчивых слайдеров и галерей  
- **AOS** — для анимаций при прокрутке элементов  
- **Google Fonts** — для кастомных шрифтов  
- **CSS Flexbox и Grid** — для адаптивной верстки  
- **Мета-теги** — для мобильной адаптивности  
- **Легкий и структурированный код** — для удобства поддержки и расширения
